Чтобы получить код создания всей вашей базы данных в pgAdmin, вы можете следовать этим шагам:

Откройте pgAdmin и подключитесь к вашему серверу базы данных.

В левой панели навигации раскройте узел "Базы данных" (Databases) и найдите вашу базу данных.

Щелкните правой кнопкой мыши на вашей базе данных и выберите "Резервное копирование..." (Backup...).

В открывшемся окне "Резервное копирование" (Backup) настройте следующие параметры:

Format (Формат): выберите "Plain" (Простой).
Encoding (Кодировка): выберите "UTF8".
Dump options (Параметры выгрузки):
Снимите флажок "Only data" (Только данные).
Установите флажок "Only schema" (Только схема).
Установите флажок "Use column inserts" (Использовать вставку столбцов).
Установите флажок "Use INSERT commands" (Использовать команды INSERT).
Оставьте остальные параметры по умолчанию.
Нажмите кнопку "Backup" (Резервное копирование).

Выберите место для сохранения файла резервной копии с расширением ".sql" и нажмите "Сохранить" (Save).

После завершения процесса резервного копирования вы можете открыть созданный файл ".sql" в любом текстовом редакторе,
чтобы просмотреть код создания вашей базы данных.

Полученный SQL-код будет содержать все инструкции, необходимые для создания структуры вашей базы данных,
включая таблицы, индексы, ограничения и другие объекты схемы.
Вы можете использовать этот код для документирования вашей схемы базы данных
или для воссоздания структуры БД в другой системе.

Надеюсь, это поможет вам получить код создания всей вашей базы данных через pgAdmin.
Если у вас возникнут дополнительные вопросы, не стесняйтесь спрашивать.
